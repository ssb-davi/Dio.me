JavaScript
- Linguagem de script (execução em tempo real) e multiplataforma
- Client Side - é executado do lado do cliente (usuário)
- Interage com elementos de uma página HTML
- Usado em desenvolvimento de páginas e também de aplicativos mobile híbridos

DOM - Document Object Model
Documento gerado pelo navegador quando carrega a página, possibilita que o
JavaScript altera todos os elementos do HTML, e os atributos e estilos de CSS
de uma página.
